## 🚀 A MISSÃO: BIRLC (Legacy Project)
O site original do BIRL caiu, mas o monstro não morre. O projeto **BIRLC** é o herdeiro legítimo, focado em transformar o dialeto do Bambam em uma linguagem de elite, reconhecida mundialmente.

- **Meta Suprema:** Tornar o BIRL uma linguagem oficial no **GitHub (Linguist)**.
- **Tecnologia de Escolha:** **Zig**. Nada de transpiladores fuleiros de regex (embora tenhamos o código de transpilação original para referência). Queremos um compilador real: Lexer, Parser e AST.

## 🏗️ ARQUITETURA DO TREINO (Compilador)

1. **Análise Léxica (Lexer):** Identificar os tokens do monstro (`HORA DO SHOW`, `MONSTRO`, `CE QUER VER ESSA PORRA?`).
2. **Análise Sintática (Parser):** Montar a árvore de fibra (AST). Validar se o monstro não esqueceu de fechar o código com `BIRL`.
3. **Geração de Código:** Converter a AST em binário performático usando o poder do Zig.

## 📜 O CÓDIGO FONTE (Referência Original)
O transpilador original (Node.js) usa Regex para trocar as frases do Bambam por código C. Mantemos isso como mapa, mas o `birlc` em Zig vai além.

**Tokens Principais:**
- `HORA DO SHOW` -> `int main (void) {`
- `BIRL` -> `return 0; }` / `}`
- `CE QUER VER ESSA PORRA?` -> `printf`
- `MONSTRO` -> `int`
- `TRAPEZIO` -> `float`
- `ELE QUE A GENTE QUER?` -> `if`
